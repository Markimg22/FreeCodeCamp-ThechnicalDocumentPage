Introdução

A interface do usuário de um site tem dois componentes. Primeiro, existem elementos visuais, como cores, fontes, imagens. Segundo, há o posicionamento desses elementos. No Web Design responsivo, um layout de interface do usuário deve acomodar muitos navegadores e dispositivos diferentes que acessam o conteúdo.

O CSS3 introduziu caixas flexíveis, ou flexbox, para criar layouts páginas para uma interface de usuário dinâmica. É um modo de layout que organiza elementos de maneira previsível para diferentes tamanhos de tela de navegadores. Embora um pouco novos, todos os navegadores modernos populares suportam o flexbox.

A adição do display: flex a um elemento o transforma em contêiner flexível.

<style> #container { display: flex; } </style> <div id="#container"></div>
Propriedade 'flex-direction'

É possível alinhar quaisquer filhos desse elemento em linhas ou colunas. Você faz isso adicionando a flex-direction propriedade ao item pai e configurando-a para linha ou coluna. Criar uma linha alinhará os filhos horizontalmente, e criar uma coluna alinhará os filhos verticalmente.

<style> #box-container { display: flex; height: 500px; flex-direction: row-reverse; ⟸ // row, column, row-reverse, column-reverse } #box-1 { background-color: dodgerblue; width: 50%; height: 50%; } #box-2 { background-color: orangered; width: 50%; height: 50%; } </style> <div id="box-container"> <div id="box-1"></div> <div id="box-2"></div> </div>
Propriedade 'justify-container'

Às vezes, os itens flexíveis em um contêiner flexível não preenchem todo o espaço no contêiner. É comum querer dizer ao CSS como alinhar e espaçar os itens flexíveis de uma certa maneira. Felizmente, a justify-content propriedade tem várias opções para fazer isso. Mas primeiro, há alguma terminologia importante a entender antes de revisar as opções.

Imagem retratando os conceitos abaixo

Lembre-se de que a configuração de um contêiner flexível como uma linha coloca os itens flexíveis lado a lado, da esquerda para direita. Um contêiner flexível definido como uma coluna coloca os itens flexíveis em uma pilha vertical de cima para baixo.

Para cada um, a direção em que os itens flexíveis são organizados é chamada de eixo principal (main axis). Para uma linha, essa é uma linha horizontal que corta cada item. E para uma coluna, o eixo principal é uma linha vertical através dos itens.

    Valores do justify-container:

  • center: alinha todos os itens flexíveis ao centro do contêiner flexível
  • flex-start: alinha itens ao ínicio do contêiner flexível.
  • flex-end: alinha itens ao final do contêiner flexível.
  • space-between: alinha os itens ao centro do eixo principal, com espaço extra colocado entre eles.
  • space-around: o espaço é distribuído em todos os itens com meio espaço em cada extremidade do contêiner flexível.
  • space-evenly: distribui o espaço uniformemente entre os itens flexíveis com um espaço total em cada extremidade do contêiner flexível.
<style> #box-container { background: black; display: flex; height: 500px; justify-content: space-between; } #box-1 { background-color: dodgerblue; width: 25%; height: 100%; } #box-2 { background-color: orangered; width: 25%; height: 100%; } </style> <div id="box-container"> <div id="box-1"></div> <div id="box-2"></div> </div>
Propriedade 'align-items'

A propriedade align-items é semelhante a justify-content. Lembre-se de que a justify-content alinhava itens flexíveis ao longo do eixo principal.

Os contêiners flexíveis também possuem um eixo transversal (cross axis), que é o oposto do eixo principal. Para linhas, o eixo transversal é vertical, e para colunas, o eixo transversal é horizontal.

O CSS oferece a align-items propriedade de alinhar itens flexíveis ao longo do eixo trasversal.

    Valores do align-items:

  • flex-start: alinha itens ao início do contêiner flexível.
  • flex-end: alinha itens ao final do contêiner flexível.
  • center: alinha itens ao centro.
  • strech: estica os itens para preencher o recipiente flexível.
  • baseline: alinha itens às linhas de base (linha em que as letras se assentam).
<style> #box-container { background: gray; display: flex; height: 500px; align-items: center; } #box-1 { background-color: dodgerblue; width: 200px; font-size: 24px; } #box-2 { background-color: orangered; width: 200px; font-size: 18px; } </style> <div id="box-container"> <div id="box-1"><p>Hello</p></div> <div id="box-2"><p>Goodbye</p></div> </div>
Propriedade 'flex-wrap'

O CSS flexbox possui um recurso para dividir um item flexível em várias linhas ou colunas. Por padrão, um contêiner flexível encaixa todos os itens flexíveis. Por exemplo, uma linha estará toda em uma linha.

No entanto, o uso da flex-wrap informa ao CSS para quebrar itens. Isso significa que itens extras são movidos para uma nova linha ou coluna. O ponto de interrupção de onde a embalagem acontece depende do tamanho dos itens e do tamanho do contêiner.

    Valores do flex-wrap:

  • nowrap: configuração padrão e não quebra itens.
  • wrap: agrupa os itens da esquerda para direita, se estiverem em uma linha, ou de cima pra baixo, se estiverem em uma coluna.
  • wrap-reverse: o inverso do wrap.
<style> #box-container { background: gray; height: 100%; display: flex; flex-direction: column; flex-wrap: wrap; } #box-1 { background-color: dodgerblue; width: 25%; height: 50%; } #box-2 { background-color: orangered; width: 25%; height: 50%; } #box-3 { background-color: violet; width: 25%; height: 50%; } #box-4 { background-color: yellow; width: 25%; height: 50%; } #box-5 { background-color: green; width: 25%; height: 50%; } #box-6 { background-color: black; width: 25%; height: 50%; } </style> <div id="box-container"> <div id="box-1"></div> <div id="box-2"></div> <div id="box-3"></div> <div id="box-4"></div> <div id="box-5"></div> <div id="box-6"></div> </div>
